# PitchLab MK2 (Export Version)

Web app that emulates the Technics SL-1200 MK2 pitch section with:
- Play / Pause (with resume)
- Real-time playback rate manipulation
- Original RPM vs playback RPM display
- Tap Tempo
- WAV export of pitched audio

Built with:
- React 19
- TypeScript
- Vite
- TailwindCSS
